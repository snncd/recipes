# recipe Rest Api 


----------------------------
## Spring Boot
---------------------------
### Tech Info

- Java 11
- Spring Boot
- Spring Data JPA
- H2 In memory database
- JUnit 5
- Lombok
- Swagger
- Maven

### Maven
```ssh
 $ cd /recipe
 $ mvn clean install
 $ mvn spring-boot:run
```
Swagger UI will be run on this url
http://localhost:8080/swagger-ui.html
